> ## 原始字面量

主要是不希望进行转义而做的声明

```c++
#include<iostream>
#include<string>
using namespace std;
int main()
{
    //使用此种写法\t会原始输出不会转义成tab
    string str= R"(D:\hello\world\test.text)";
    cout << str << endl;
    return 0;
}
```
